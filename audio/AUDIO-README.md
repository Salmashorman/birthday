# ملفات الصوت المطلوبة

حط ملفات الـ mp3 هون بنفس الأسماء بالضبط (حروف صغيرة، بدون مسافات):

| اسم الملف         | الوصف                                              | نوع التشغيل |
|--------------------|-----------------------------------------------------|--------------|
| `wind.mp3`         | صوت رياح ليلية خفيفة                                | loop (متكرر) |
| `howl.mp3`         | عواء ذئب واحد، ثانيتين تقريبًا                       | مرة واحدة عند الدخول |
| `forest.mp3`       | أصوات غابة خفيفة (طيور ليلية، حشرات...)              | loop (متكرر) |
| `ambient.mp3`      | موسيقى هادئة بدون كلمات ترافق المشهد                | loop (متكرر) |
| `favorite.mp3`     | الأغنية المفضلة (تشتغل فقط عند الضغط على الزر)       | loop عند التشغيل |

## ملاحظات
- إذا اسم أي ملف مختلف عندك، بس افتح `index.html` وغيّر قيمة `src` بالسطر المطابق:
  ```html
  <audio id="sfxWind" src="assets/audio/wind.mp3" loop></audio>
  ```
- ما في حد إجباري تحطه كلهم — أي ملف ناقص، الموقع رح يتجاهله بهدوء وما رح يظهر أي خطأ للزائر.
- حجم الملفات: يفضل تكون خفيفة (mp3 مضغوط بجودة متوسطة) حتى ما يبطّئ تحميل الصفحة، خصوصًا `ambient.mp3` و`favorite.mp3` إذا كانت أغنية كاملة.
